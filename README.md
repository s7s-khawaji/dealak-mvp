# Dealak MVP 🚀
## Real Estate Demand Marketplace

Dealak is a real estate demand marketplace that connects serious property buyers with suitable real estate brokers.  
Unlike traditional real estate listing platforms, Dealak focuses on customer requests:  
Customer creates a request → Brokers receive the request → Brokers show interest → Customer chooses the suitable broker.

---

# 🎯 MVP Goal
This MVP is built to validate three main assumptions:
1. Do customers create real estate requests?
2. Do brokers engage with customer requests?
3. Are brokers willing to pay for access to serious customers?
